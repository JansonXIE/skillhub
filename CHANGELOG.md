# 更新日志 (CHANGELOG)

## [0.1.0] - 2026-05-10

### 新增
- **从 GitHub 安装技能**: 
  - 在新建技能弹窗中增加了“从 GitHub 安装”的专属交互视图。
  - 支持直接在前端输入 GitHub 仓库 URL，调用 Tauri 核心 API 在后台触发下载。
  - 在 Rust 后端新增 `clone_github_repo` 命令，默认通过 `git clone` 自动将仓库获取到本地的 `../skills` 目录下。
- **新建技能弹窗**: 实现了全新的“新建技能”对话框，支持多种添加方式。
  - **AI 草稿**: 支持通过描述需求由 AI 生成 SKILL.md。
  - **从 GitHub 安装**: 支持粘贴仓库地址进行安装。
  - **手动创建**: 支持从零开始编写技能。
  - **扫描本地**: 支持导入已有的本地技能。
- **状态管理**: 在顶部栏（Header）中集成了弹窗控制逻辑。

### 修复
- **运行时错误**: 修复了由于 `lucide-react` 缺少 `Github` 图标导致的页面白屏问题，暂时使用 `GitBranch` 替代。
- **编译优化**: 清理了 `Header.tsx` 中未使用的导入，解决了严格模式下的构建失败。
- **UI 样式**: 
  - 移除了弹窗关闭按钮及顶部栏按钮多余的黑色边框。
  - 为弹窗添加了平滑的淡入和位移动画。
  - 优化了弹窗在深色模式下的适配效果。
